# Gerenciador de Geladeira

Esta é uma aplicação web simples, desenvolvida em **HTML5**, **CSS3** e **JavaScript**, que facilita o controle de validade e ajuda a reduzir o desperdício de alimentos na geladeira.

## Funcionalidades

- Cálculo automático da validade dos alimentos em uma geladeira.
- Listagem de produtos vencidos e em bom estado.
- Indicação do estado dos produtos listados com cores.
- Informação precisa das datas de adição e vencimento dos produtos.

## Como usar

Para usar, [clique aqui](https://arthurnsc.github.io/gerenciador-de-geladeira/) ou clone o repositório e abra o arquivo `index.html` em um navegador.
